# Planning activités condensé

Le planning d'activités liste toutes les activités des camps et des réservations pour une tranche horaire à une date.

## Où le trouver ?

Apps dashboard → Réservations → Planning → Activités

## Finalité métier

Cette vue sert de tableau de pilotage quotidien ou hebdomadaire pour les équipes terrain.

Elle permet d'avoir sur un même écran :

- les activités déjà planifiées ;
- la répartition par tranche horaire ;
- une lecture mixte des besoins issus des réservations et des camps.

## Lecture fonctionnelle

Le planning condensé est particulièrement utile quand il faut :

- vérifier les conflits d'occupation sur une tranche horaire ;
- préparer les intervenants et ressources ;
- arbitrer les changements de dernière minute.

## Lien avec les documents

Le package contient aussi un document d'impression d'activités de réservation (`print-booking-activity`) qui reprend les activités, les repas et les groupes d'un séjour sous une forme distribuable. Le planning condensé est donc la vue de pilotage à l'écran, tandis que le document d'activités est la restitution imprimable.
