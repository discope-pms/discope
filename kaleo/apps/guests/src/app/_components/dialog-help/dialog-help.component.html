Bienvenue sur la fiche d'hébergement de votre réservation.

Cette application vous permet de renseigner les identités des personnes reprises dans votre réservation.
Pour vous aider dans l'encodage, plusieurs indicateurs vous renseignent sur l'état de la fiche:

  • Finalisation : un pourcentage vous indique le niveau de complétion de votre fiche ("Finalisation : x%", sous le bouton "Soumettre").
  • Validité : si certaines informations nécessaires sont manquantes pour soumettre la fiche, le bouton "Soumettre" reste désactivé.

Si votre réservation inclut plusieurs séjours, chaque séjour aura sa propre section avec les dates correspondantes.

Pour chaque séjour, il faut, au minimum, une personne désignée comme responsable (vous pouvez en désigner plusieurs si nécessaire). Cela signifie qu'une fiche peut être soumise, même si tous les participants ne sont pas renseignés.

Pour pouvoir soumettre la fiche, il faut que, pour chaque ligne, tous les champs soient remplis (à l'exception du NISS pour les enfants de moins de 15 ans).
Si vous ne disposez pas de toutes les informations obligatoires pour compléter une ligne, vous devrez supprimer celle-ci pour pouvoir soumettre la fiche.

Après l'envoi de vos informations, la fiche reste accessible, en lecture seule, jusqu'à la fin de votre réservation.
Un lien "Exporter" vous permet de télécharger la liste des participants au format XLS.
