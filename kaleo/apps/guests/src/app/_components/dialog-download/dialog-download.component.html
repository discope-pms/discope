Téléchargement de votre fiche d’hébergement.

Le lien ci-dessous vous permet de télécharger un fichier XLS contenant la liste des informations d’hébergement telles que vous les avez soumises.

Vous pouvez utiliser ce fichier pour consultation ou, si besoin, pour formuler une demande de modification.

Dans le cas d'une modification, toute demande doit être soumise pour validation au gîte pour lequel la réservation a été prise, sous réserve de son acceptation.

Avertissement : Kaleo se réserve le droit de refuser toute demande de modification. Une modification soumise pourrait donc ne pas être acceptée.
