# Descriptif applicatif

Cette section decline le fonctionnement de Kaleo dans Discope par grands domaines fonctionnels :

- catalogue, tarification et regles commerciales;
- reservations, contrats, financements et plans de paiement;
- channel manager, caisse, comptabilite et suivi operationnel.

Les pages ont ete decoupees a partir du document source afin de rendre les sujets navigables independamment.
