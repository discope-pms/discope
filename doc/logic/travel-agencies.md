## Tour-Opérateurs

Dans le cas d'un séjour via tour-opérateur, le tarif est celui de
l'organisation, mais une commission, convenue à l'avance, doit être
reversée à un partenaire « tour-opérateur ».

Dans le cas d'une réservation via Tour-Opérateur, le client est le TO
(contrats et factures émis à leur nom.), et l'hôte est un contact de la
réservation.

Il y a une liste de partenaires (Tour-Opérateurs) : à chaque TO peut
être associé un taux de commission.

Lors de la facturation, si le payeur est un TO, ajouter une réduction
correspondant au taux de commission sur la facture.

Dans le cas d'un montant fixe payé par le partenaire, une facture
distincte est créée pour la réservation, adressée au partenaire (la
référence est le code de la réservation concernée).

## Réservations via Tour-Opérateur (TO / OTA)

Lors d'une réservation via un Tour Opérateur, une commission est due à
l'opérateur sur la réservation. Sauf exception, les OTA sont gérées
comme des TO dans Discope.

Cette commission peut être payée de différentes façons :

-   l'hôte paye le séjour: un pourcentage du montant facturé doit être
    versé au TO comme commission

-   le TO paye le séjour: la facture est envoyée au TO avec une
    réduction pour commission (peut ne s'appliquer que sur les nuitées)

Dans le cadre de réservations via TO, il est pertinent de prévoir un
pack avec le TO pour lequel le prix est convenu d'avance (comprenant
les types de chambres ou les unités locatives spécifiques, avec
suppléments éventuels).

Dans tous les cas, l'organisation doit être notifié qu'il s'agit
d'une réservation "Tour Opérateur" (soit via le TO, soit via le
bénéficiaire), puisqu'il faut pouvoir identifier les réservations pour
lesquelles une commission est due.

De manière générale, c'est la personne (morale ou physique) qui paie le
séjour qui est considérée comme le Client (et au nom de qui la
réservation doit être prise).
